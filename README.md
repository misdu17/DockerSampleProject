# DockerSampleProject

This repository contains WebDriver code examples, exercises and tutorials for developers. Over time, more and more test example will be uploaded here.

## Pre-requisites

You'll need to install:

1. Java 8.
2. Maven.
3. Firefox.
4. Chrome.
